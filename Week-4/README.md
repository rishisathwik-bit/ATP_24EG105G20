# Week 4 – Backend Development with Node.js and Express.js

## Overview

Week 4 focuses on backend development using Node.js, Express.js, and MongoDB. The main objective was to build RESTful APIs,
connect the backend with a database, and understand server-side application architecture.

---

## Topics Covered

- Introduction to Node.js
- Express.js Framework
- REST API Development
- Middleware
- MongoDB Basics
- CRUD Operations
- Routing
- Server Setup

---

## Technologies Used

- Node.js
- Express.js
- MongoDB
- Mongoose
- JavaScript (ES6)
- Nodemon
