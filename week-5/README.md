# Week 5 – Authentication and Authorization using JWT

## Overview

Week 5 focuses on implementing secure authentication and authorization in the MERN stack application using JSON Web Tokens (JWT).
The objective was to create secure login systems, protect routes, and manage user roles such as USER and AUTHOR.

---

## Topics Covered

- User Authentication
- Authorization
- JSON Web Token (JWT)
- Password Hashing
- Middleware
- Protected Routes
- Role-Based Access Control
- Login and Registration APIs

---

## Technologies Used

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT (jsonwebtoken)
- bcryptjs
- React.js
- Axios
