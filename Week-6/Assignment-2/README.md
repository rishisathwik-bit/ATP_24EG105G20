# Week 6 – Assignment 2

## Overview

Week 6 Assignment 2 focuses on advanced frontend-backend integration in a MERN stack application. The assignment demonstrates secure API communication,
authentication handling, protected routes, state management, and debugging techniques for real-world full-stack applications.

This assignment strengthened understanding of API integration, role-based authorization, token handling, and frontend state
management while solving common deployment and communication issues.


# Project Objectives

The primary objectives of this assignment are:

- Build a full-stack MERN application
- Integrate frontend and backend systems
- Implement JWT authentication
- Manage application state
- Handle protected routes
- Perform CRUD operations
- Debug authentication and API issues
- Improve frontend-backend communication
