# Week 6 – Assignment 1

## Overview

Week 6 Assignment 1 focuses on integrating frontend and backend applications in a MERN stack environment. The project demonstrates how React frontend
applications communicate with Node.js and Express.js backend APIs using Axios, while implementing authentication, authorization, protected routes, and state management.

This assignment helped in understanding real-world full-stack application workflows including API integration, JWT authentication, state persistence, 
and debugging frontend-backend communication issues.


# Project Objectives

The main objectives of this assignment are:

- Integrate frontend and backend systems
- Implement authentication flow
- Consume REST APIs using Axios
- Manage global state
- Handle protected routes
- Debug API communication issues
- Understand full-stack architecture
