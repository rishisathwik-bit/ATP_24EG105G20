# Week 1 - JavaScript Practice Folder

This folder contains JavaScript practice files for Week 1 of the ATP 24EG105G20 course. The exercises focus on core programming concepts such as arrays, functions, classes, and simple algorithms.

## Files Included
- `Array.js` - Array operations and examples.
- `ArrayFunction.js` - Functions that work with arrays.
- `class.js` - Class definition and object usage in JavaScript.
- `EmployeesArray.js` - Employee data stored and processed in arrays.
- `MaxOfThree.js` - Finds the maximum of three values.
- `maxOfTwo.js` - Compares two values and returns the maximum.
- `MinOfArray.js` - Finds the minimum value in an array.
- `sumofmarks.js` - Calculates sum totals, likely for marks or scores.

## Purpose
This folder is used to practice JavaScript logic and exercise basic programming skills, including:
- Working with arrays
- Writing reusable functions
- Using classes and objects
- Performing conditional comparisons
- Summing and analyzing numeric data

## Notes
- Run these scripts using Node.js or in a browser console.
- Each file is self-contained and demonstrates a small JavaScript concept or problem.
