# Week 2 – React Basics and Component Development

## Overview
This week focuses on learning the fundamentals of React.js and modern JavaScript concepts required for frontend development. 
The tasks involve creating reusable components, understanding JSX syntax, and building interactive user interfaces using React and Vite.

Repository Link: 
https://github.com/rishisathwik-bit/ATP_24EG105G20/tree/main/Week-2

## Topics Covered

- Introduction to React.js
- React Project Setup using Vite
- JSX Syntax
- Functional Components
- Props and State
- ES6 Features
- Event Handling
- Component Reusability
